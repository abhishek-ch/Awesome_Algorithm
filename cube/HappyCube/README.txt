Open the package com.cube.test.

There are 3 test cases -
a)Challenge1Part1Test - Runs Part a of challenge 1
b)Challenge1Part2Test - Runs Part b of challenge 1
c)Challenge2Test - Runs the challenge 2

Run any of the test case above , it will create a directory named output. Inside output directory, you will find
all given Cube name colored directory. Under each directory you will find each test case linked sub directory..
